## 前端工具库

提供高质量的无依赖的前端基础工具

每个文件完成一件事,尽可能无依赖,考虑全面.

可直接使用es6,es7语法,无需对新语法层面做兼容

每个封装都要有说明和示例,写到代码注释里

只有 `src` 目录下的文件被发布到`npm`
